# DevilConnection_ModManager

使用 Vuetify CLI 脚手架搭建。

## ❗️ 文档

* 主要文档：[https://vuetifyjs.com/](https://vuetifyjs.com/)
* 入门指南：[https://vuetifyjs.com/en/getting-started/installation/](https://vuetifyjs.com/en/getting-started/installation/)
* 社区支持：[https://community.vuetifyjs.com/](https://community.vuetifyjs.com/)
* 问题追踪：[https://issues.vuetifyjs.com/](https://issues.vuetifyjs.com/)

## 🧱 技术栈

* 框架：Vue 3 + Vite
* UI 库：Vuetify
* 语言：TypeScript
* 包管理器：npm

## 🧭 从这里开始

* 主入口：`src/main.ts`
* 主应用组件：`src/App.vue`
* 主要样式：`src/styles/`
* 插件设置：`src/plugins/`

## 📁 项目结构

* `src/main.ts` — 应用程序入口
* `src/App.vue` — 根组件
* `src/components/` — 可复用的 Vue 组件
* `src/plugins/` — 插件注册与配置
* `src/styles/` — 全局样式与主题设置
* `public/` — 静态公共文件

## ✨ 已启用功能

* ESLint
* Pinia
* Vue Router
* UnoCSS + Vuetify 预设

## 💿 安装

使用您选择的包管理器 (npm) 安装依赖：

```bash
npm install
```

## 🚀 快速启动

```bash
npm install
npm run dev
```

## 🏗️ 构建

```bash
npm run build
```

## 🧪 可用脚本

* `npm run dev`
